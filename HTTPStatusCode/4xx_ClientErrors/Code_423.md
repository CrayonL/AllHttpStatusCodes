# HTTP状态码 423 (Locked) 含义详解

---

## HTTP 423 状态码

HTTP 423 (Http Status Code 423) 状态是HTTP协议的一种响应码，是我们请求访问网站时，服务器端返回的4xx 客户端错误系列响应码之一。

---

**状态码含义：**  
**HTTP423状态码代表的意思是** **当前资源被锁定**，即 **HTTP 423 Locked** 响应状态。

**状态详细说明：**  
**【http code 423】** 表示当前资源被锁定。（RFC 4918 WebDAV）
